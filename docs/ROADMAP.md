# Roadmap

## Milestone 1: Public MVP

- Resume text input.
- Job description input.
- Match score.
- Extracted matching skills.
- Missing skills.
- Learning roadmap.
- FastAPI endpoint.
- React UI.
- Unit tests and GitHub Actions.

## Milestone 2: Resume Uploads

- PDF upload.
- Text extraction.
- Resume section parsing.
- Better skill taxonomy.
- Exportable report.

## Milestone 3: ML and NLP Upgrade

- Sentence-transformer semantic similarity.
- Role-specific scoring weights.
- Model evaluation dataset.
- MLflow experiment tracking.
- Bias and reliability notes.

## Milestone 4: Data Engineering Layer

- PostgreSQL storage.
- Application tracking table.
- Batch analysis pipeline with Airflow or Prefect.
- dbt-style analytics models.

## Milestone 5: MLOps and Cloud

- Docker image publishing.
- Cloud deployment.
- Monitoring and health checks.
- CI/CD deployment workflow.
- Release tags and changelog.
