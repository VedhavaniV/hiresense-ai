# Open Source Hosting Guide

## Recommended Platforms

- **GitHub:** main repository, issues, project board, releases.
- **GitHub Container Registry:** Docker images.
- **Vercel or GitHub Pages:** frontend demo.
- **Render, Fly.io, Railway, or Hugging Face Spaces:** backend demo.
- **Hashnode, Dev.to, or Medium:** project case study.

## Repository Setup Checklist

- Add MIT license.
- Add README with screenshots.
- Add CONTRIBUTING guide.
- Add good first issues.
- Add architecture diagram.
- Add project board.
- Add first release tag after MVP.
