# Contributing to HireSense AI

Thank you for helping improve HireSense AI.

## Good First Contributions

- Add more skills to the skill taxonomy.
- Improve resume parsing examples.
- Add tests for edge cases.
- Improve UI accessibility.
- Add deployment guides for open-source-friendly platforms.
- Write documentation for students and freshers.

## Local Development

1. Fork the repository.
2. Create a feature branch.
3. Make a focused change.
4. Add or update tests.
5. Open a pull request with a clear description.

## Pull Request Checklist

- The change has a clear purpose.
- Tests pass locally.
- Documentation is updated if behavior changed.
- No secrets, API keys, or private data are committed.
